# Real-Time Helmet & No Helmet Detection AI

A custom-trained Computer Vision AI system built using YOLOv8 and OpenCV for real-time helmet and no-helmet detection.

---

# Technologies Used

- Python
- OpenCV
- YOLOv8
- PyTorch
- NumPy
- Google Colab
- Roboflow

---

# Project Structure

```bash
ComputerVisionAI/
│
├── models/
│   └── best.pt
│
├── src/
│   └── custom_helmet_detection.py
│
├── videos/
│   └── helmet_test.mp4
│
├── screenshots/
│
├── requirements.txt
├── README.md

```

---

# Download Model

Download the trained YOLOv8 model (`best.pt`) from the link below and place it inside the `models/` folder.

## Model Download Link

https://drive.google.com/file/d/1aA3IQn8eqs9j2T9yoRP_m5ZotTQJMiuF/view?usp=drive_link

---

# Install Required Libraries

Run this command inside the project folder:

```bash
pip install -r requirements.txt
```

---

# Run The Project

Run the following command:

```bash
python src/custom_helmet_detection.py
```
